# Configuration
To run this project, follow these steps:

1. **How to Run**:
   - Install project dependencies:
     ```bash
     yarn install
     ```
   - renam env.example to .env :
     ```bash
    VITE_APP_HOST="http://localhost:3001"
     ```
     ```bash
    VITE_APP_WS_HOST="ws://localhost:3001"
     ```
     
2. **Start Services**:
   - Start project :
     ```bash
    yarn dev
     ```
